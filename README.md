## Launch setup

Install and set `zsh`

```
sudo apt update && sudo apt install -y zsh
sudo chsh -s "$(which zsh)"
```

Install `oh-my-zsh`

```
sh -c "$(curl -fsSL https://raw.github.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
```

Clone

```
git clone https://github.com/Leyka/pi.git && cd pi
```

Run

```
./install.sh
```
